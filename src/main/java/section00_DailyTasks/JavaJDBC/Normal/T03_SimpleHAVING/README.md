Дана таблица `people`:

`____________people___________
| id  |	  name  | age | ... |
| 1   |	Alex    | 17  | ... |
.............................
| 15  | Cicero  | 24  | ... |`

Посчитай, сколько людей имеют одинаковый возраст, и верни группы с 2 или более людьми, имеющими этот возраст.

Результат должен содержать два столбца: `age` и `total_people`.
